# RecSim_FlashcardLearning
## Testing
### Environments
 - Python 3.10
 - tensorflow 2.14.0
 - dopamine 4.0.6
 - recsim 0.2.4

Note that you can just use `pip install recsim` to collect all of these dependencies, but some libraries may not work properly without these versions.

### Execution
`python main.py`
